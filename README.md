# PaperAI

AI-powered exam paper generator for Indian teachers.

## 🚀 Features
- Generate exam papers in one click
- Supports CBSE / State Boards
- Choose class, subject, and exam type
- AI-generated structured question papers

## 🛠 Built With
- Lovable
- AI tools
- Web technologies
